# Example Usage Scripts

This directory contains example scripts demonstrating how to use the Email Management Assistant API.

## Prerequisites

Make sure the API server is running:
```bash
python main.py
```

## Examples Included

1. `fetch_and_analyze.py` - Fetch emails and analyze them with AI
2. `send_email.py` - Send an email
3. `classify_spam.py` - Check emails for spam

## Running Examples

```bash
# Make sure you're in the virtual environment
source venv/bin/activate

# Run an example
python examples/fetch_and_analyze.py
```

## Note

Remember to configure your `.env` file with proper email credentials before running these examples.
